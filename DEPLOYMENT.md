# 7th Grade Math Tutor - Setup & Deployment Guide

## Quick Setup

### On-Premise (Local)
```bash
cd ghc-ex1
python -m http.server 8000
# Open http://localhost:8000
```

### Vercel Deployment
```bash
npm install -g vercel
vercel
# Follow prompts
```

## Teacher Dashboard Access
Click "Teacher Dashboard" button on main page to:
- Create classes
- Add students
- Create assignments
- View reports
- Share on WhatsApp

## All features work both locally and on Vercel (free hobby tier)
